## 🧠 Detalhes do Curso Atual

### **Curso 1: Como usar IA no fluxo real de desenvolvimento**

Neste primeiro passo da formação, o objetivo é transformar a teoria em prática real de engenharia de software, elevando o nível de automação e qualidade do código através das seguintes competências:

* Uso do Claude-Code para automatizar a criação de boilerplate, endpoints e testes automatizados.
* Implementação de pipelines de CI/CD e automações com GitHub Actions para integração, revisão e pull requests automatizados.
* Revisão critica do código produzido por IA aplicando padrões de design, testes e práticas de segurança.
* Integração de ferramentas de design como Figma via MCP.
* Desenvolvivmento de estratégias de testes automatizados e validação contínua para garantir qualidade em produtos duráveis.
